# Restaurant Order Billing System

This is a command-line based Restaurant Order Billing System in Python.

## Features

- Display Menu
- Add Items to Order
- Auto-generate Order ID
- Calculate Total, Tax (5%), and Discount (10% on bills over ₹500)
- Display formatted Bill Summary

## Project Structure

```
restaurant/
├── __init__.py
├── menu.py
├── order.py
├── bill.py
├── offers.py
main.py
```

## How to Run

```bash
python main.py
```

Enjoy ordering!
